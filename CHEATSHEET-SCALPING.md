# 🎯 CHEATSHEET SCALPING — SVP + SnR + proScalper

> **Filosofi inti:** SnR & SVP = **OTAK** (di mana entry). proScalper = **JARI** (kapan pencet).
> Jangan pernah tukar posisi keduanya. Level dulu, sinyal belakangan.

---

## ⚖️ 5 ATURAN EMAS (baca tiap sebelum entry)

1. **Bias M15 dulu.** Bullish → cari BUY saja. Bearish → cari SELL saja. Konflik → DIAM.
2. **Entry HANYA di level.** SnR zone / VAH / VAL / POC. Di tengah kosong (*no man's land*) = HARAM entry, walau label nyala.
3. **Level itu titik keputusan, bukan sinyal.** Tunggu reaksinya: **REJECT (mantul)** atau **BREAK + RETEST**.
4. **POC = magnet.** Selalu jadi target pertama (TP1). Harga cenderung balik ke POC.
5. **Jangan chase.** Kalau momentum udah "kebuka lebar" & harga jauh dari level = kamu telat. Tunggu retest.

---

## 🗺️ PETA HARGA (Value Area SVP)

```
   PREMIUM (mahal)   ── di atas VAH ──   ← zona JUAL / target akhir BUY
  ══════════ VAH ══════════   ← batas atas value  (resist / TP / titik reversal)
   UPPER VALUE
  ────────── POC ──────────   ← MAGNET (TP1 selalu ke sini)
   LOWER VALUE
  ══════════ VAL ══════════   ← batas bawah value (support / TP / titik reversal)
   DISCOUNT (murah)  ── di bawah VAL ──   ← zona BELI / target akhir SELL
```

**Confluence = level lebih kuat:** kalau **VAH/VAL/POC nyambung dengan zona SnR** → itu level grade-A. Prioritaskan.

## 🔀 ALUR KEPUTUSAN (ikutin urut, jangan loncat)

```
1) M15 Bias?          BULLISH → cari BUY   |  BEARISH → cari SELL   |  konflik/NO TRADE → DIAM
2) Harga di LEVEL?    tidak (kosong) → DIAM |  ya → lanjut
3) Reaksi di level?   REJECT → entry arah pantulan  |  BREAK+RETEST → entry arah break  |  belum jelas → TUNGGU
4) proScalper label?  label muncul + M5 Trend searah + Sesi AKTIF → ENTRY  |  kalau tidak → TUNGGU
5) Kelola:            TP1=POC → TP2=VA edge → SL=balik ke level. BE di POC.
```

## ⛔ KAPAN HARUS DIAM (NO TRADE)

| Kondisi | Kenapa |
|---------|--------|
| Dashboard = **NO TRADE** / M15 vs M5 konflik | Arah nggak jelas → whipsaw |
| Harga di **tengah value** (jauh dari VAH/VAL/POC & SnR) | *No man's land*, nggak ada level pegangan |
| Label **lawan bias M15** (tanpa reversal confluence kuat) | Lawan arus institusi = kalah |
| Dashboard **Sesi = TUTUP** | Jam sepi, likuiditas tipis |
| Harga **di level tapi reaksi belum jelas** | Belum ketahuan reject/break → sabar |
| Momentum **udah kebuka lebar**, harga jauh dari level | Entry telat = ketangkep pantulan |

---

## 🎓 REJECT vs BREAK (ini yang paling sering bikin bingung)

**Di sebuah level (misal VAH pas turun bearish):**

- **REJECT (mantul):** muncul candle balik arah + wick nolak level → level **nahan**. Entry **searah pantulan**, TAPI hanya kalau searah bias M15.
- **BREAK + RETEST:** candle **close nembus** level → tunggu harga **balik nyentuh level dari sisi lain** (level flip fungsi) → entry **searah break**. Ini entry paling aman.
- **Belum jelas:** candle masih kebentuk / nggak ada penolakan/penembusan tegas → **TUNGGU**, jangan nebak.

> 🔑 **Jangan SELL di support (VAL/VAH-as-support). SELL setelah support JEBOL & di-RETEST.**
> 🔑 **Jangan BUY di resist. BUY setelah resist JEBOL & di-RETEST.**

## 💰 MANAJEMEN (jangan dilanggar)

- **TP1 = POC** (magnet). Amankan sebagian di sini, geser SL ke **breakeven**.
- **TP2 = VA edge** (VAH untuk BUY, VAL untuk SELL) atau SnR berikutnya.
- **SL selalu di balik level** yang jadi alasan entry (di balik wick rejection / di balik zona flip).
- **Risk/Reward minimal 1:1.5.** Kalau POC terlalu dekat (RR jelek) → skip.
- Maksimal loss beruntun → berhenti, jangan revenge trading.
