🎯 Entry Playbook — SnR + SVP

Scalping XAU · USDJPY · EURUSD · GBPUSD · Sesi malam (mulai 19:00 WIB) · Workflow M15 → M5 → M1

PRINSIP INTI: SnR = DI MANA harga bereaksi (zona + diamond ◆). SVP = SEBERAPA VALID level itu (POC/VAH/VAL). Entry terbaik = dua-duanya menunjuk harga yang sama (confluence).

📊 Peta Sinyal

SnRBox merah + ◆ merah (atas)Resistance hold → bias SELL

Prefix p = level dari sesi sebelumnya (prior-day), mis. pPOC = naked POC kemarin.

🪜 Alur Eksekusi

  • BIAS 🧭 — tentukan arah

    Lihat zona SnR utama + posisi harga terhadap POC & prior-day (pPOC/pVAH/pVAL). Harga di atas value area & mantul support = bias BUY. Di bawah & tolak resistance = bias SELL. Bias tidak jelas → SKIP.

  • TREND 📈 — konfirmasi arah

    Struktur searah bias M15 (higher-low untuk buy / lower-high untuk sell). Pastikan ada zona SnR M5 yang nyambung dengan level SVP (VAL/VAH/POC). Ini yang menaikkan kelas setup.

  • ENTRY 🎯 — eksekusi presisi

    Tunggu harga retest level confluence tadi. Trigger = ◆ diamond muncul + candle rejection (pin/engulf). Masuk saat candle trigger close.

✅ Checklist Entry

CHECKLIST BUY
  • M15 bias bullish (di atas / mantul VAL–support)
  • Confluence: box support SnR ≈ VAL / pVAL / pPOC
  • M1: ◆ hijau + candle rejection
  • RR minimal 1:2 ke target (POC / VAH berikutnya)
CHECKLIST SELL
  • M15 bias bearish (di bawah / tolak VAH–resistance)
  • Confluence: box resistance SnR ≈ VAH / pVAH / pPOC
  • M1: ◆ merah + candle rejection
  • RR minimal 1:2 ke target (POC / VAL berikutnya)

🏅 Kualitas Setup

Grade A — size normal · SnR diamond + level SVP + searah bias M15. → yang diburu.
Grade B — size kecil · diamond ada tapi SVP netral.
Grade C — SKIP · diamond M5/M1 sendirian, tanpa SVP & tanpa bias M15 → ±70% fake.

⏰ Timing

  • Sesi malam mulai 19:00 WIB (≈ London sore / NY) — likuiditas bagus untuk XAU & major pairs.
  • Entry terbaik 30–45 menit setelah level M15 terbentuk.
  • Retest pertama = win-rate tertinggi. Retest ketiga biasanya jebol.
