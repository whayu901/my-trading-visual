🎯 CHEATSHEET SCALPING

SVP + SnR + proScalper  |  M15 = arah · M5 = konfirmasi · M1 = eksekusi

Filosofi inti: SnR & SVP = OTAK (di mana entry). proScalper = JARI (kapan pencet). Level dulu, sinyal belakangan.

⚖️ 5 Aturan Emas

  1. Bias M15 dulu. Bullish → cari BUY saja. Bearish → cari SELL saja. Konflik → DIAM.
  2. Entry HANYA di level. SnR / VAH / VAL / POC. Di tengah kosong (no man's land) = HARAM entry, walau label nyala.
  3. Level = titik keputusan, bukan sinyal. Tunggu reaksinya: REJECT (mantul) atau BREAK + RETEST.
  4. POC = magnet. Selalu jadi target pertama (TP1). Harga cenderung balik ke POC.
  5. Jangan chase. Momentum udah kebuka lebar & harga jauh dari level = kamu telat. Tunggu retest.

🗺️ Peta Harga (Value Area)

PREMIUM (mahal) di atas VAH ← zona JUAL / target BUY ══════════ VAH ══════════ batas atas value UPPER VALUE ────────── POC ────────── MAGNET (TP1 ke sini) LOWER VALUE ══════════ VAL ══════════ batas bawah value DISCOUNT (murah) di bawah VAL ← zona BELI / target SELL

Confluence: kalau VAH/VAL/POC nyambung dengan zona SnR → level grade-A. Prioritaskan.

🔀 Alur Keputusan

1) M15 Bias?
   BULLISH → cari BUY
   BEARISH → cari SELL
   konflik → DIAM

2) Harga di LEVEL?
   kosong → DIAM
   ya → lanjut

3) Reaksi di level?
   REJECT      → entry arah pantulan
   BREAK+RETEST→ entry arah break
   belum jelas → TUNGGU

4) proScalper label + M5 searah
   + Sesi AKTIF → ENTRY  |  else TUNGGU

5) TP1=POC → TP2=VA edge
   SL = balik ke level · BE di POC

💰 Manajemen

  • TP1 = POC → amankan sebagian, SL ke breakeven.
  • TP2 = VA edge (VAH utk BUY, VAL utk SELL) / SnR berikut.
  • SL selalu di balik level alasan entry.
  • RR minimal 1 : 1.5. POC kedekatan → skip.
  • Loss beruntun → STOP, jangan revenge.

🎓 REJECT vs BREAK

  • REJECT (mantul): candle balik arah + wick nolak level → level nahan. Entry searah pantulan (kalau searah bias M15).
  • BREAK + RETEST: candle close nembus level → tunggu balik nyentuh dari sisi lain (level flip) → entry searah break. Paling aman.
  • Belum jelas: candle masih kebentuk / nggak ada penolakan tegas → TUNGGU.
🔑 Jangan SELL di support. SELL setelah support JEBOL & di-RETEST.
🔑 Jangan BUY di resist. BUY setelah resist JEBOL & di-RETEST.
